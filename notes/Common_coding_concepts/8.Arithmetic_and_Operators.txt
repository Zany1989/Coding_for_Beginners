8. Arithmetic and Operators.

Как вы, возможно, помните из нашего задания по программированию, быстрая арифметика была одной из главных причин
изобретения компьютеров. Поэтому неудивительно, что код может помочь нам в решении математических задач.

Мы можем легко использовать код для сложения, вычитания, умножения и деления чисел, и вы увидите, что большинство языков
используют одни и те же символы. Например, в Python и JavaScript мы используем + для сложения, - для вычитания, * для
умножения и / для деления.

Конечно, операторы используются не только для чисел. Например, мы можем использовать оператор сложения для сложения
строк. Здесь мы сложили строку "hello world" со строкой "hello world", получив новую строку: "Hello world".

Мы также можем задавать вопросы типа: больше ли X, чем Y? Или это одно и то же? Для этого мы используем операторы
сравнения. Например, в Python и JavaScript мы используем > для проверки, больше ли X, чем Y, и == для проверки, равны
ли X и Y. Меньше или равно 40 — это меньше или равно 40. Итак, 40 <= 40 — это правда, а 40 <= 39 — это ложь. Не
равно 25 — это не равно 20.5 — это правда, а 25 != 25 — это ложь.

Сравнение может быть очень полезным для принятия решений в наших программах, например, в условных операторах и
циклах. Операторы сравнения возвращают логические значения (Boolean), которые могут быть использованы для управления
потоком выполнения программы.

Мы можем пойти ещё дальше, используя логические операторы, чтобы задавать вопросы типа: «X и Y больше, чем Z?» В Python
языке включают and, or и not. Оператор and возвращает True, если оба операнда истинны, оператор or возвращает True,
если хотя бы один из операндов истинен, а оператор not возвращает противоположное значение логического операнда.
Например, (X > Z) and (Y > Z) будет возвращать True, если оба X и Y больше Z, а (X > Z) or (Y > Z) будет возвращать
True, если хотя бы один из них больше Z.

Так почему же нельзя использовать один знак равенства? В большинстве языков программирования один знак равенства
используется для присваивания значений данным, а не для проверки. Например, в языке Python, если вы хотите проверить,
равны ли X и Y, вы должны использовать двойной знак равенства (==). Один знак равенства (=) используется для
присваивания значения переменной. Например, X = 5 присваивает значение 5 переменной X.
