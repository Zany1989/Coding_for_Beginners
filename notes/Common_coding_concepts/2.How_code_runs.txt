2. How code runs?

Другой термин для этого — выполнение. Выполнение кода означает запуск вашего кода на компьютере в исполняемом файле.
Файл — это файл или программа, содержащая ваш код и доступная для запуска компьютером.

Как правило, весь код читается слева направо, символ за символом, сверху вниз. Построчно или пошагово. Мы можем дать
машине команду остановиться, вернуться назад и снова выполнить определенные строки кода, а также пропустить или
проигнорировать определенные строки, не выполняя эти инструкции.

Инженеры-программисты очень заботились о скорости выполнения своего кода и измеряли её. Фактически, если существовало
несколько разных способов написания кода для достижения одного и того же результата, большинство инженеров выбирали
самый быстрый или наиболее эффективный метод, потому что он был на миллисекунды быстрее другого.

Скорость действительно становится очень важной в определённых приложениях, например, в финансовом секторе, где скорость
транзакций, происходящих по всему миру, имеет решающее значение для кода в этих системах. Они готовы были бы отдать всё,
чтобы получить дополнительную миллисекунду от своего кода.

IDE — интегрированная среда разработки. Она получила свое название потому, что содержит все необходимое программисту
для написания, тестирования и запуска кода в одном месте.
